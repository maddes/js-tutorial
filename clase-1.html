¿Qué corno es un script Javascript?

Bueno, un archivo Javascript es una secuencia de comandos que la compu entiende y se ejecutan ordenadamente.

Por ejemplo, el siguiente script:

			document.write('Hola')
			document.write('que')
			document.write('tal')
			document.write('como')
			document.write('te')
			document.write('va')

Dará como resultado lo siguiente:

Un horror, todo junto.
Pero bueno, cada sentencia se ejecutó EXACTAMENTE después de la otra, dando como resultado eso.
Es tu culpa no haberle puesto espacios. Bueno, mía en realidad.

Intentá editar el código y refrescar la página como para arreglarlo y que se vea lindo.
También podés probar cambiar el orden, etc… Jugar un rato.

(Como viste, document.write() simplemente muestra lo que le pidas entre paréntesis)
